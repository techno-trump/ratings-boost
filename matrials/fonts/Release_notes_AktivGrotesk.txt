Technology
----------

File formats:
    - Static fonts: 
        - Desktop: TTF (TrueType-based OpenType format)
        - Apps: TTF (TrueType-based OpenType format)
        - Webfonts: WOFF; WOFF2 (Web Open Font Format). Subset webfonts provided for Corp, Arabic, Thai, Hebrew, Georgian, Armenian, Devanagari, Malayalam character sets.     

    - Variable: 
        - Desktop: TTF (TrueType-based OpenType format)
        - Apps: TTF (TrueType-based OpenType format)
        - Webfonts: WOFF; WOFF2 (Web Open Font Format). Subset webfonts provided for Corp, Arabic, Thai, Hebrew, Georgian, Armenian, Devanagari, Malayalam character sets.  

Screen optimization information: 
- PixelFix CTAA screen optimization technology.

Default numerals: 
    - Tabular for Latin, Devanagari and Malayalam. Proportional numerals accessible via OpenType features (pnum feature)
    - Proportional for Arabic and Hebrew. Tabular numerals accessible via OpenType features (tnum feature)
    - Proportional only for Thai.

Other Information
----------

The fonts contain the following OpenType Stylistic Sets:
    - ss01: Urdu numbers (Arabic)
    - ss02: Latin numbers (Arabic, Devanagari, Malayalam and Thai)
    - ss03: Vowel insertion above name of God (Arabic)
    - ss04: Alternate UC Greek prosgegrammeni forms (Greek)
    - ss10: Disable Dutch ij ligatures

The fonts contain the following OpenType features
    - locl: Localised forms
    - ordn: Ordinals
    - frac/sups/subs/sinf/numr/dnom: Fractions and small numerals
    - pnum/tnum: Proportional and Tabular numerals
    - lnum/onum: Lining numerals and Old Style numerals
    - smcp: Small Caps and numerals (Latin, Cyrillic and Greek)
    - c2sc: Caps to Small Caps (Latin, Cyrillic and Greek)
    - unic: Unicase (All Small Caps with alternate a e m n u). All European Latin only.
    - liga: Standard ligatures (Latin and Armenian)
    - rlig: Required Ligatures (Arabic)
    - case: Case Punctuation
